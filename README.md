# Очередь в первый класс

Простая веб-страница для отслеживания позиции в очереди на зачисление в первый класс по нескольким школам одновременно.

## Как пользоваться

Откройте `index.html` в браузере или задеплойте как статику на любой хостинг.

Введите номер заявления в поле «Номер заявления» — он сохранится в localStorage и подставится при следующем открытии. Номер также прописывается в URL-параметр `?target=...`, ссылкой можно поделиться.

## Источники данных

- **Количество мест** в каждой школе — берётся с Госуслуг вручную и прописывается в массив `schools` в коде.
- **Заявления и распределение по группам** — загружаются в реальном времени с сайта Министерства образования Свердловской области: [minobraz.midural.ru](https://minobraz.midural.ru/activity/3060/)

## Группы очереди

Очередь разбита на четыре группы в порядке приоритета:

1. **Внеочередники** — дети с особым правом вне очереди
2. **Первоочередники** — дети с первоочередным правом
3. **Братья и сёстры** — дети, чьи братья/сёстры уже учатся в школе
4. **Без льгот** — общая очередь

На диаграмме 100% соответствует количеству мест в школе. Серый сегмент — свободные места.

## Свой форк

Хотите отслеживать другие школы? Всё просто:

1. Сделайте форк репозитория
2. Найдите нужные школы на [minobraz.midural.ru](https://minobraz.midural.ru/activity/3060/) и скопируйте их `id` из URL запроса
3. Обновите массив `schools` в `index.html`:

```js
const schools = [
  { name: "Школа 200", id: "eee15c16-...", capacity: 120 },
  { name: "Ваша школа", id: "ваш-id",     capacity: 100 },
];
```

Никаких зависимостей, сборки и сервера не нужно — один HTML-файл. Легко деплоится на [GitHub Pages](https://pages.github.com/): достаточно включить Pages в настройках репозитория, указав ветку `main` и корневую папку.
